# FabFlix

## About
Fabflix is a buy/search movies web application.

## Back-End
-Java\
-Grizzly(Http Server)\
-JAX-RS(RESTful Web Services)\
-JDBC(Database Connectivity)\
-PayPal API

## Front-End
-JavaSript\
-React.js\
-Axios(Http Client)\
-ReactBootstrap\
-Redux/React Redux

## Database
-MySQL
